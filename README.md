BiometricalSolutions
====================

Biometrical Solitions (Fingerprint Verification, Face Recognition)

Biometric Interface Developer – Video/Image Acquisition, Fingerprint Scanners, MRZ Passport Reader, Barcode Readers.
Development of API to support a broad range of scanners including fingerprint scanners,  passport readers, barcode readers, document scanners.
Development of DLL library of functions for video stream processing support using DirectX technology.
 - Fingerprint, barcode/passport reader proprietary SDKs.
 - Neurotechnology fingerprint verification SDK (VeryFinger, VeryLook), WSQ Fingerprint image library
 - Microsoft DirectX 9 API
 - TWAIN image acquisition API
 - Microsoft Visual Studio 2008, Microsoft SQL Server Compact Edition
 - C#, Visual C++ V6.0
